# circlecheck

同人イベントでの利用を想定した、 javascript 製の簡易的なリスト表示アプリです。
決められた仕様の json データを読み込んでリスト表示をします。

## 機能

- json の読み込み
- 読み込んだデータをタブ形式で分類して一覧表示
- 項目にチェック（お気に入り機能）
- お気に入りに対してのチェック機能

## 設置方法

サーバー側にロジックは必要ありませんので、任意のフォルダに展開するだけで動きます。（ json データは任意で利用する必要があります。）

## 依存ライブラリ

circlecheck は以下のライブラリを使用しています

- jQuery
- Bootstrap
- Hogan.js
- Holder.js
